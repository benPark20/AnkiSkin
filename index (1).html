⚠️ GitHub Pages + HTTPS

When served over https://, browsers block requests to AnkiConnect's http://localhost endpoint (mixed-content policy).

Fix A (recommended): In your repo → Settings → Pages, uncheck Enforce HTTPS. Access the app via http://username.github.io/repo.

Fix B: Add your Pages URL to AnkiConnect's webCorsOriginList, then in Chrome open Site Settings → Insecure content → Allow for this page.
